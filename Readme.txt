Build and Test Instructions

1. Install Apache Ant from https://ant.apache.org/bindownload.cgi

2. Navigate to the project directory.

3. To clean the project:
   ant clean

4. To compile the project:
   ant compile

5. To run the tests:
   ant test

6. To create a deployable package:
   ant package

The compiled code will be in the 'build' directory, and the deployable package will be 'devops_repo.tar.gz'.
